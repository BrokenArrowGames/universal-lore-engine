A tool for worldbuilding with a focus on writing book, video game, and TTRPG stories.

## Installation
```bash
pnpm install
```

## Running the app
```bash
# development
pnpm start:debug

# production mode
pnpm start
```

## Test
```bash
# unit tests
pnpm test

# e2e tests
pnpm test:e2e

# test coverage
pnpm test:cov
```
